{% filter markdown %} # About P7PP P7PP, codename `web-pp`, is a Web port of [`alfred-pp`][alfred-pp], a workflow for [Alfred][alfred] on Mac OS X which allows one to quickly go on a teacher's page using a few keystrokes. Data is fetched from the Web only once, and then stored locally. This implementation allows anyone to use `pp`. It works exactly the same way as the original one. [alfred-pp]: https://github.com/bfontaine/alfred-pp#readme [alfred]: http://www.alfredapp.com/ ## How to use it Start typing a teacher's name in the box. It'll suggest you some names. The first one is highlighted by default, and you can move using up and down arrows. Press enter to open the selected teacher's personal page. Additionally, pressing ctrl and enter at the same time opens the page in a new tab instead of using the current one (you can use shift if you prefer, or cmd on OS X). Press tab to replace your input with the selected teacher's name. You can also use the good ol' mouse, but the goal is to save your time, this is why the box has already the focus when you open P7PP. See more on the [wiki][]. [wiki]: https://github.com/bfontaine/web-pp/wiki ## How it works The application is written in Python (using Flask and Gunicorn), and uses Redis to store some info. Every night around 4AM, it fetches LIAFA, PPS and Gallium member lists, extract some info from them (at least each name and webpage URL), and store them locally. It also performs some lexical transformations to facilitate search operations later. When you go on the app, it serves you the most up to date people list and the all the code it needs. The search is done on the client side to avoid unecessary requests. The JavaScript code uses [AngularJS][ng] for the suggestions rendering and [Mousetrap][mt] for keyboard shortcuts. All the code is agressively compressed to speed up loading time. Everything is available on [GitHub][ppgh].
